Email Alerts

Email alerts are configured via GitHub repository secrets. Go to your repository settings to update.

SMTP_HOST SMTP server hostname
SMTP_USER SMTP username/email
SMTP_PASS SMTP password or app password
ALERT_RECIPIENTS Comma-separated emails for your team
